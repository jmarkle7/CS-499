Database Enhancement

CS 340 Animal Rescue Dashboard

Enhancement Overview

This enhancement focused on improving the CS 340 Animal Rescue Dashboard through expanded database filtering and improved data interaction. Additional filters were implemented to allow users to search by rescue type, breed, age, sex, and animal type. A matching record count was also added to provide immediate feedback regarding query results. These enhancements improved usability while demonstrating database design, query optimization, and data management concepts.
