Database Enhancement

CS 340 Animal Rescue Dashboard

Enhancement Overview

This enhancement focused on improving the CS 340 Animal Rescue Dashboard through expanded database filtering and improved data interaction. Additional filters were implemented to allow users to search by rescue type, breed, age, sex, and animal type. A matching record count was also added to allow users to see how many records met their queries. These enhancements improved usability by allowing users to search through 10,000 records with more filtering options.

Skills Demonstrated

  • MongoDB Query Development
  • Data Filtering Techniques
  • Data Management
  • User Interface

Course Outcome Alignment

This enhancement demonstrates my ability to create or enhance software to meet industry related goals. By adding advanced filtering, the client or user will be able to search for records faster to find or analyze their data.
