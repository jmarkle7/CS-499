Software Engineering Enhancement

CS 360 Weight Tracking Application

Enhancement Overview

This enhancement involves creating a new "Goal Tracking" screen for users to be able to visualize their progress. Components of this screen include a progress bar, and a message prompted to the user. There are also stats that show the user's starting weight, current weight, goal weight, and the pounds remaining until their goal weight is reached.

Skills Demonstrated

  • Software Engineering
  • Mobile Application Development
  • UI Design
  • Testing/Debugging
  • Goal Tracking Logic
  • Progress Calculation

Course Outcome Alignment

This enhancement demonstrates my ability to solve a given problem using algorithmic principles and computer science practices and standards appropriate to its solution while managing the trade-offs involved in design choices. By adding a screen with goal tracking and progress logic, it solves the problem for user's not being able to see their progress. There was a trade-off in design, however; the application is still easily accessible for users.
